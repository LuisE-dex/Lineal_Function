# Linear Function Calculator

Aplicación desarrollada en Java Script que calcula la ecuación de una función lineal a partir de dos puntos proporcionados por el usuario.

Dado un par de puntos `(x₁, y₁)` y `(x₂, y₂)`, el programa calcula la pendiente y la ordenada al origen para obtener la ecuación de la recta en la forma:

[
y = mx + n
]

## Características

* Cálculo automático de la pendiente (`m`).
* Cálculo de la ordenada al origen (`n`).
* Generación de la ecuación de la recta.

## Ejemplo

Entrada:

```text
Punto 1: (1, 3)
Punto 2: (5, 11)
```

Salida:

```text
y = 2x + 1
```
